# 正規表現入門

正規表現の基本的なパターンを解説します。

## 基本パターン

- `.` - 任意の1文字
- `*` - 0回以上の繰り返し
- `+` - 1回以上の繰り返し

```javascript
const regex = /\d{3}-\d{4}/
regex.test('123-4567') // true
```

## まとめ

正規表現をマスターして文字列処理を効率化しましょう。
